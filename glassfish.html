Instalación y configuración de Glassfish en Eclipse

Requisitos necesarios antes de comenzar:

1.- Antes de comenzar con la instalación del servidor Glassfish, es necesario tener instalado el jdk, en mi caso estoy utilizando el jdk1.7.0



2.- Además de tener en nuestra PC, el IDE Eclipse para Java EE Developers (Es muy importante que Eclipse sea para un entorno Web). Si no lo tienen, les dejo el link donde pueden descargar el IDE for Java EE Developers.

3.- Descargar el Servidor de Aplicaciones Glassfish (En la imagen se muestra la opión a descargar en caso que estés utilizando un entorno Windows)



Comenzando la instalación:

Vamos a la carpeta donde se ha descargado el servidor, damos clic derecho en el setup y lo ejecutamos con permisos de administrador.


El asistente se iniciará y aparecerán las siguientes pantallas.



La primera pantalla nos da una pequeña introducción sobre el servidor Glassfish, sólo damos clic en Siguiente.



Seleccionamos el tipo de Instalación típica.



Dejamos que el servidor se instale en el directorio por defecto y damos clic en Siguiente.



Desactivamos la casilla de Instalar Herramienta de Actualización y damos clic en Siguiente.



Si todo va bien, damos clic en Instalar.



Progreso de la instalación.



En esta ventana lo único que vamos a establecer es la contraseña de nuestro servidor, en este caso será 12345. Puedes poner la que desees, siempre y cuando no se te olvide.



Se muestra un resumen de los resultados de la configuración, sólo damos clic en Siguiente.



Una vez finalizada la instalación, damos clic en Salir.



Configurando Glassfish en Eclipse

Muy bien, una vez abierto Eclipse, nos vamos al menu Window > Preferences



En la siguiente ventana, damos clic en Server > Runtime Environment y damos clic al botón Add...

Ahora damos clic en Download additional server adapter.



Buscamos Oracle Glassfish Server Tools y damos clic.



Posteriormente, damos clic en Next.



Aceptamos los términos y damos clic en Finish.



Aparecerá una ventana que nos indica que el servidor Oracle Glassfish Server Tools será descargado e instalado, nos avisará cuando la instalación finalice para reiniciar Eclipse. Damos clic en OK.



En la parte inferior derecha del IDE, se muestra que el servidor se está descargando.



Una vez que la instalación haya finalizado, se solicitará que se reinicie Eclipse, damos clic en Restart Now.

Una vez abierto Eclipse, regresamos a la ventana de configuración del servidor. (Window > Preferences > Server > Runtime Environment > Add...).

En la ventana seleccionamos la carpeta Glassfish - Glassfiss 3.1.2, y en la parte inferior de la ventana damos clic a la casilla Create a new local server y clic en Next.



Para configurar el runtime, glassfish requiere que tengamos configurado el jdk para ello seleccionamos el link Installed JRE preferences.



En la siguiente ventana, damos clic en Add...



En la ventana Add JRE seleccionamos la opción Standard VM y damos clic en Next.



En la ventana siguiente, debemos especificar el directorio del jdk que tenemos instalado, presionamos el botón Directory... y buscamos la ruta, en este caso es C:\Program Files\Java\jdk1.7.0 y clic en Finish.


En la ventana siguiente, debemos especificar el directorio del jdk que tenemos instalado, presionamos el botón Directory... y buscamos la ruta, en este caso es C:\Program Files\Java\jdk1.7.0 y clic en Finish.


Seleccionamos la casila del jdk que hemos agregado y damos clic en OK.


En la opción JRE seleccionamos jdk1.7.0, y para la opción Application Server Directory debemos ingresar el path donde esta instalado nuestro glasssfish, en este caso es C:\glassfish3\glassfish y damos clic en Finish.



En la siguiente ventana definimos la contraseña que pusimos cuando instalamos el servidor: 12345, y dejamos los valores de domain Directory y Admin name por defecto.



En la siguiente ventana, solo damos clic en OK, y listo, hemos configurado nuestro servidor glassfish.



Para verificar que funciona nuestro servidor, nos vamos a la pestaña Servers en la parte inferior de Eclipse y para iniciarlo damos clic derecho y seleccionamos Start.



El estado cambiará a [Started, Synchronized]



Para realizar alguna configuración del servidor sólo hay que dar doble clic y se despliega una ventana para editar las configuraciones.



Puedes verificar que Glassfish haya iniciado, abriendo tu navegador y tecleando localhost:4848 , ingresas el usuario: admin y la contraseña: 12345 (o la que hayas establecido).
